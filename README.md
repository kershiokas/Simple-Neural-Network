Simple Neural Network
-
About:
if you are using this, good luck :)
Made this to learn more about neural networks.

What it can do?
-
- Matrix creation with common matrix operations:
  scalar, matrix and hadamard multiplication, matrix addition/subtraction and transposition.
- Other Matrix functions: print, randomize and apply_Function (for activations).
- Activation functions: sigmoid and relu (including their derivatives).
- Layers support forward pass.
- Network supports forward and backward pass.
- Save and load trained weights :D (it's literally just a text file though)


Output:

0 XOR 0 = 0.013

0 XOR 1 = 0.989

1 XOR 0 = 0.990

1 XOR 1 = 0.009
